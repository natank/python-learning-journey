## 📋 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- Python (v3.8 or higher)
- npm and pip package managers

### Installation
```bash
# Install Node.js dependencies
npm install

# Install Python dependencies
cd apps/flask-guess-game && pip install -r requirements.txt && cd ../..
```

### Development Commands

#### Web Servers
```bash
# Start Express server (http://localhost:3000)
npm run express-server

# Start Flask server (http://localhost:5000)
npm run flask-guess-game
```

#### Python Development
```bash
# Run Python exercises
npm run python-beginner-exercises

# Run test suite
npm run test-python
```
